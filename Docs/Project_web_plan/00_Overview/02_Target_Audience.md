## 1. 오디언스 개요 | Audience Overview

### 1.1 1차 타겟 (Primary Audience)

```
┌─────────────────────────────────────────────────────────────┐
│                     1차 타겟 그룹                            │
├─────────────────────────────────────────────────────────────┤
│  🏛️ 교회 리더십          │  👨‍👩‍👧‍👦 기독교 가정           │
│  • 목사/전도사           │  • 부모                       │
│  • 장로/권사             │  • 가정 예배 인도자           │
│  • 교육부 담당자         │  • 자녀 신앙교육 관심자       │
├─────────────────────────────────────────────────────────────┤
│  👤 개인 신자            │  📚 소그룹 리더               │
│  • 성경공부 열심자       │  • 셀그룹/순 리더             │
│  • 은사/사역 탐색자      │  • 제자훈련 인도자            │
│  • 영적 성장 추구자      │  • 양육 담당자                │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 2차 타겟 (Secondary Audience)

### 6.2 언어/톤 선호도

```
교회 리더:     신학적 깊이 + 실용성 ──────── 격식체, 전문 용어 OK

청년 개인:     영감 + 도전 ────────────── 직접적, 공감적
```
