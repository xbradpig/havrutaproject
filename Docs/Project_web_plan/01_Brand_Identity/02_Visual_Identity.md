## 1. 디자인 철학 | Design Philosophy

### 1.1 비주얼 원칙

Havruta Project의 비주얼 아이덴티티는 브랜드 철학을 시각적으로 표현합니다.

```
┌─────────────────────────────────────────────────────────────┐
│                     비주얼 원칙                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  1. 깊이 (Depth)                                            │
│     - 표면적 화려함이 아닌 깊이감                           │
│     - "더 깊은 뿌리"를 시각적으로 표현                      │
│                                                              │
│  2. 연결 (Connection)                                       │
│     - 개별 요소들이 유기적으로 연결                         │
│     - 공동체와 관계를 시각화                                │
│                                                              │
│  3. 뿌리 (Root)                                             │
│     - 견고하고 안정적인 기반                                │
│     - 흔들리지 않는 정체성                                  │
│                                                              │
│  4. 빛 (Light)                                              │
│     - 어둠 속 진리의 빛                                     │
│     - 희망과 방향성                                         │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 디자인 키워드

| 키워드 | 시각적 표현 |
|--------|-------------|
| Timeless | 트렌디하지 않지만 시대를 초월하는 |
| Grounded | 안정적이고 견고한 |
| Warm | 차갑지 않은, 따뜻한 |
| Connected | 연결되고 유기적인 |
| Purposeful | 의도적이고 의미있는 |

## 6. 이미지 스타일 | Image Style

### 6.1 사진 스타일

```
PHOTOGRAPHY STYLE
─────────────────────────────────────────────────────────────

분위기:
- 따뜻하고 자연스러운
- 연출되지 않은 진정성
- 빛이 아름다운 시간대 (골든아워)
- 깊이감 있는 구도

주제:
- 함께하는 모습 (공동체)
- 성경/말씀 관련 이미지
- 가정/가족 모임
- 자연 요소 (나무, 뿌리, 빛)

색감:
- 따뜻한 톤 (Warm)
- 자연스러운 채도
- 부드러운 대비

피해야 할 것:
✗ 과도하게 연출된 포즈
✗ 스톡 사진 느낌
✗ 차갑거나 기업적인 톤
✗ 과도한 필터
```

### 6.2 일러스트레이션 스타일

```
ILLUSTRATION STYLE
─────────────────────────────────────────────────────────────

스타일:
- 미니멀하고 의미있는
- 선형(Line-based) 또는 플랫
- 브랜드 컬러 활용
- 개념을 시각화

용도:
- 부흥 루프 다이어그램
- 프로젝트 생태계 맵
- F.A.M.I.L.Y 원칙 시각화
- 아이콘 확장 그래픽

특징:
- 단순하지만 깊은 의미
- 연결성 강조
- 유기적인 형태
```

### 6.3 그래픽 요소

```
GRAPHIC ELEMENTS
─────────────────────────────────────────────────────────────

1. 부흥 루프 (Revival Loop)
   - 순환하는 화살표
   - 6개 단계 표현
   - 애니메이션 가능

2. 뿌리/나무 모티프
   - 깊이와 성장 표현
   - 배경 패턴 활용

3. 빛 모티프
   - 그라데이션 광선
   - 강조점 표현

4. 연결선
   - 점선/실선
   - 요소 간 관계 표현
```

## 9. 애니메이션 가이드 | Animation Guidelines

### 9.1 모션 원칙

```
MOTION PRINCIPLES
─────────────────────────────────────────────────────────────

1. 목적성 (Purposeful)
   - 장식이 아닌 의미있는 모션
   - 사용자 행동에 대한 피드백

2. 자연스러움 (Natural)
   - ease-out 기반 이징
   - 급격한 변화 지양

3. 일관성 (Consistent)
   - 동일한 요소는 동일한 애니메이션
   - 예측 가능한 동작

4. 성능 (Performance)
   - transform, opacity 위주
   - will-change 적절히 활용
```

### 9.2 애니메이션 속성

| 속성 | 값 | 사용 |
|------|-----|------|
| Duration (빠름) | 150ms | 버튼 hover, 토글 |
| Duration (보통) | 250ms | 카드 hover, 드롭다운 |
| Duration (느림) | 400ms | 페이지 전환, 모달 |
| Easing | cubic-bezier(0.4, 0, 0.2, 1) | 기본 |
| Easing (입장) | cubic-bezier(0, 0, 0.2, 1) | 요소 등장 |
| Easing (퇴장) | cubic-bezier(0.4, 0, 1, 1) | 요소 사라짐 |

### 9.3 스크롤 애니메이션

```
SCROLL ANIMATIONS
─────────────────────────────────────────────────────────────

Fade Up:
  From: opacity 0, translateY(20px)
  To: opacity 1, translateY(0)
  Duration: 500ms
  Trigger: 요소가 뷰포트 진입 시

Stagger:
  각 요소 100ms 딜레이
  리스트, 그리드 아이템에 적용

Parallax:
  속도 차이로 깊이감 표현
  Hero 배경, 장식 요소에 제한적 사용
```
