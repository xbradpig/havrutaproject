## 2. 핵심 메시지 체계 | Core Message Framework

### 2.1 핵심 메시지 계층

```
CORE MESSAGE HIERARCHY
─────────────────────────────────────────────────────────────

Level 1: Brand Promise (브랜드 약속)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Level 2: Value Propositions (가치 제안)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

3. 공동체 연결 - "함께함으로 서로를 지킵니다"
4. 지속 가능한 부흥 - "이벤트가 아닌 루프를 세웁니다"

Level 3: Supporting Messages (지원 메시지)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 3. 대조 메시지 | Contrast Messages

### 3.1 세상 vs Havruta

```
CONTRAST MESSAGES
─────────────────────────────────────────────────────────────

세상이 말합니다:                우리는 말합니다:
"더 똑똑해져라"          →     "더 깊은 뿌리를 내려라"
"경쟁에서 살아남아라"    →     "서로를 지켜라"
"답을 찾아라"            →     "진리를 사랑하는 마음을 만들어라"
"혼자서도 할 수 있다"    →     "함께할 때 더 강하다"
"성능으로 가치를 증명해라" →   "존재 자체가 가치다"
```

### 3.2 사용 예시

| 위치 | 세상의 방식 (X) | Havruta의 방식 (O) |
|------|----------------|-------------------|
| Hero | — | "더 깊은 뿌리로 세워지는 교회" |
| About | — | "성능이 아닌 형상으로 인간을 정의합니다" |
| ChurchThrive | "성장 해킹" | "지속 가능한 부흥의 구조" |
| COMPASS | "경쟁력 강화" | "은사를 발견하고 사역을 찾다" |
