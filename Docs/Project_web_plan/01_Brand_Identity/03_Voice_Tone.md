## 1. 브랜드 보이스 | Brand Voice

### 1.1 보이스 정의

**Havruta Project의 목소리**는 진실되고 따뜻하며, 깊이 있으면서도 접근 가능합니다.

```
┌─────────────────────────────────────────────────────────────┐
│                    BRAND VOICE                               │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  "우리는 함께 말씀 앞에 서서,                               │
│   서로의 이야기를 듣고,                                      │
│   진리를 향해 걸어가는 동반자입니다."                        │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 보이스 속성

| 속성 | 설명 | 예시 |
|------|------|------|
| **진실된 (Authentic)** | 꾸밈없이 솔직하게 | "쉽지 않습니다. 하지만 함께라면 가능합니다." |
| **따뜻한 (Warm)** | 차갑지 않고 환영하는 | "당신의 여정에 함께하게 되어 기쁩니다." |
| **깊이 있는 (Deep)** | 표면적이지 않은 | "더 깊은 뿌리가 더 큰 열매를 맺습니다." |
| **겸손한 (Humble)** | 교만하지 않은 | "우리도 배워가는 중입니다." |
| **희망적 (Hopeful)** | 긍정적이지만 현실적인 | "어둠이 깊을수록 빛은 더 빛납니다." |

### 1.3 보이스 스펙트럼

```
VOICE SPECTRUM
─────────────────────────────────────────────────────────────

격식적 ─────────────●─────────────────── 캐주얼
                    ↑
                우리 위치: 따뜻하지만 진지한

기술적 ────────────────●──────────────── 대중적
                       ↑
                   우리 위치: 접근 가능한 깊이

권위적 ──────────●────────────────────── 친근한
                 ↑
             우리 위치: 신뢰할 수 있지만 다가가기 쉬운

학술적 ─────────────●─────────────────── 실용적
                    ↑
                우리 위치: 의미 있으면서 적용 가능한
```

## 2. 톤 가이드 | Tone Guide

### 2.1 톤의 정의

**톤**은 상황에 따라 보이스가 표현되는 방식입니다. 보이스는 일관되지만, 톤은 맥락에 따라 조절됩니다.

### 2.2 상황별 톤

#### Landing Page - 영감을 주는 (Inspiring)

```
✓ 비전을 제시
✓ 감정에 호소
✓ 강렬하지만 공격적이지 않음
✓ 희망과 가능성

예시:
"ASI 시대, 우리가 지켜야 할 것은 경쟁력이 아닙니다.
 정체성입니다."
```

#### About Page - 스토리텔링 (Storytelling)

```
✓ 내러티브 형식
✓ 감정적 연결
✓ 여정을 함께하는 느낌
✓ 철학을 이야기로 전달

예시:
"어느 날, AI가 모든 답을 제시하는 시대에도
 한 작은 공동체는 이렇게 모여 앉아 있었습니다..."
```

#### Project Pages - 명확한 (Clear)

```
✓ 정보 중심
✓ 구체적이고 실용적
✓ 기능과 혜택 설명
✓ 행동 유도

예시:
"ChurchThrive 15일 부트캠프는 교회가 부흥의 구조를
 세우도록 돕는 집중 프로그램입니다."
```

#### Resources - 가이드하는 (Guiding)

```
✓ 친절하고 단계적
✓ 실용적인 도움
✓ 격려하는 톤
✓ 질문에 답하는 형식

예시:
"가정예배가 처음이신가요? 괜찮습니다.
 함께 첫 걸음을 떼어볼까요?"
```

#### Contact - 환영하는 (Welcoming)

```
✓ 열린 자세
✓ 경청하는 태도
✓ 응답을 약속
✓ 따뜻한 초대

예시:
"궁금한 점이 있으시면 언제든 물어보세요.
 우리는 여러분의 이야기를 듣고 싶습니다."
```

## 3. 작성 원칙 | Writing Principles

### 3.1 핵심 원칙

```
┌─────────────────────────────────────────────────────────────┐
│                   WRITING PRINCIPLES                         │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  1. 말씀에 뿌리를 둔다                                       │
│     - 성경적 기반이 있는 메시지                              │
│     - 인용 시 정확한 출처                                    │
│                                                              │
│  2. '우리'로 함께한다                                        │
│     - 가르치는 것이 아닌 함께 걷는 것                        │
│     - "해야 합니다" → "함께 해봅시다"                        │
│                                                              │
│  3. 깊이와 접근성의 균형                                     │
│     - 의미를 희석하지 않으면서                               │
│     - 누구나 이해할 수 있게                                  │
│                                                              │
│  4. 실용적 적용                                              │
│     - 이론에서 멈추지 않고                                   │
│     - 구체적인 다음 단계 제시                                │
│                                                              │
│  5. 희망을 전달                                              │
│     - 문제를 직시하되                                        │
│     - 해결책과 희망을 함께                                   │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 구체적 가이드라인

#### ✓ 이렇게 씁니다

| 카테고리 | Good Example |
|----------|--------------|
| 시작 | "함께 시작해볼까요?" |
| 설명 | "말씀을 통해 우리는 ..." |
| 제안 | "이렇게 해보시는 건 어떨까요?" |
| 격려 | "작은 시작이 큰 변화를 만듭니다." |
| 문제 인식 | "쉽지 않다는 걸 압니다." |

#### ✗ 이렇게 쓰지 않습니다

| 카테고리 | Bad Example | 이유 |
|----------|-------------|------|
| 명령적 | "해야 합니다" | 강압적 |
| 판단적 | "이것만이 정답입니다" | 교만함 |
| 추상적 | "영적 성장을 하세요" | 구체성 부족 |
| 복잡한 | 긴 문장, 전문 용어 남발 | 접근성 저하 |
| 부정적 | "이러면 안 됩니다" | 희망 부재 |

## 4. 언어 가이드 | Language Guide

### 4.1 한국어 작성 가이드

```
한국어 작성 원칙
─────────────────────────────────────────────────────────────

문장:
- 한 문장에 하나의 생각
- 평균 20-30자 내외
- 수동태보다 능동태
- 명사형 종결 자제

어휘:
- 순우리말 우선 (가능한 경우)
- 신학 용어 사용 시 설명 병기
- 외래어는 최소화

존칭:
- 기본: '~입니다', '~습니다' (해요체 아님)
- 친근함 필요 시: '~해요' 가능
- '여러분', '당신' 적절히 사용

피해야 할 표현:
✗ "~하셔야 합니다" (명령적)
✗ "~인 것 같습니다" (불확실)
✗ "많은" (구체적 수치 선호)
```

### 4.2 영어 작성 가이드

```
English Writing Guidelines
─────────────────────────────────────────────────────────────

Sentence Structure:
- Short and clear sentences
- Active voice preferred
- Avoid jargon
- One idea per sentence

Word Choice:
- Simple over complex
- Concrete over abstract
- Familiar over technical
- Strong verbs over adverbs

Tone:
- Conversational but respectful
- "We" over "You should"
- Warm and inviting
- Confident but humble

Avoid:
✗ "You must..." (commanding)
✗ "Obviously..." (condescending)
✗ Passive voice when active is possible
✗ Filler words (very, really, just)
```

### 4.3 이중 언어 일관성

| 요소 | 한국어 | English |
|------|--------|---------|
| 톤 | 동일한 따뜻함 | Same warmth |
| 의미 | 번역이 아닌 재창작 | Recreate, don't translate |
| 문화적 맥락 | 한국 교회 문화 반영 | Global church context |
| 성경 인용 | 개역개정 | ESV/NIV |

## 5. 콘텐츠 유형별 가이드 | Content Type Guide

### 5.1 헤드라인/제목

```
HEADLINE GUIDELINES
─────────────────────────────────────────────────────────────

Hero 헤드라인:
- 강렬하고 시적
- 질문 또는 선언
- 8단어 이내
- 예: "더 깊은 뿌리로 세워지는 교회"

섹션 제목:
- 명확하고 직접적
- 내용을 예고
- 6단어 이내
- 예: "부흥은 루프입니다"

카드/리스트 제목:
- 구체적이고 실용적
- 행동 중심
- 4단어 이내
- 예: "가정예배 시작하기"
```

### 5.2 본문 텍스트

```
BODY TEXT GUIDELINES
─────────────────────────────────────────────────────────────

단락:
- 3-4문장이 이상적
- 하나의 아이디어
- 자연스러운 흐름

문장:
- 다양한 길이 (단문과 복문 혼합)
- 접속사로 자연스럽게 연결
- 읽기 쉬운 리듬

강조:
- Bold: 핵심 개념
- Italic: 특별한 용어, 인용
- 과도한 강조 지양
```

### 5.3 CTA (Call-to-Action)

```
CTA GUIDELINES
─────────────────────────────────────────────────────────────

Primary CTA:
- 동사로 시작
- 명확한 혜택 암시
- 2-4단어
- 예: "지금 시작하기", "무료 가이드 받기"

Secondary CTA:
- 부담 적은 행동
- 탐색/발견 강조
- 예: "더 알아보기", "사례 보기"

피해야 할 것:
✗ "여기 클릭" (불명확)
✗ "제출" (차가움)
✗ 너무 긴 CTA 텍스트
```

### 5.4 마이크로카피

```
MICROCOPY GUIDELINES
─────────────────────────────────────────────────────────────

폼 레이블:
- 명확하고 간결
- 예: "이메일 주소", "전화번호"

플레이스홀더:
- 도움이 되는 힌트
- 예: "예: ministry@church.com"

에러 메시지:
- 친절하고 해결책 제시
- 예: "이메일 형식을 확인해주세요."

성공 메시지:
- 따뜻하고 격려하는
- 예: "감사합니다! 곧 연락드리겠습니다."

빈 상태:
- 긍정적이고 안내하는
- 예: "아직 저장된 항목이 없습니다. 첫 번째를 추가해볼까요?"
```

## 6. 피해야 할 표현 | Expressions to Avoid

### 6.1 금지 표현

```
EXPRESSIONS TO AVOID
─────────────────────────────────────────────────────────────

교만한 표현:
✗ "우리만이 답을 알고 있습니다"
✗ "다른 방법은 틀렸습니다"
✗ "이것이 유일한 길입니다"

판단적 표현:
✗ "제대로 된 신앙인이라면..."
✗ "진정한 크리스천은..."
✗ "믿음이 부족해서..."

공포 마케팅:
✗ "지금 안 하면 늦습니다"
✗ "실패할 것입니다"
✗ "뒤처질 것입니다"

과장된 표현:
✗ "혁명적인"
✗ "완벽한 솔루션"
✗ "모든 문제를 해결"

모호한 표현:
✗ "영적 성장"만 (구체화 필요)
✗ "축복받으세요"만 (구체화 필요)
✗ "은혜로운" (구체화 필요)
```

### 6.2 대안 표현

| 피해야 할 표현 | 대안 |
|----------------|------|
| "해야 합니다" | "함께 해보면 어떨까요" |
| "실패하지 마세요" | "성공의 길을 함께 걸어요" |
| "혁명적인" | "새로운 접근" |
| "완벽한" | "도움이 되는", "효과적인" |
| "모두가 알아야 할" | "함께 나누고 싶은" |
| "간단합니다" | "차근차근 시작할 수 있어요" |

## 7. 성경 인용 가이드 | Scripture Citation Guide

### 7.1 인용 원칙

```
SCRIPTURE CITATION PRINCIPLES
─────────────────────────────────────────────────────────────

1. 정확성
   - 구절을 정확히 인용
   - 문맥을 존중
   - 출처 명시

2. 적절성
   - 메시지와 자연스럽게 연결
   - 억지로 끼워맞추지 않음
   - 핵심을 강조하는 용도

3. 접근성
   - 인용 후 설명 병기 (필요시)
   - 긴 구절은 핵심 부분만
   - 현대어 번역 선호
```

### 7.2 인용 형식

```
인용 형식 (한국어)
─────────────────────────────────────────────────────────────

짧은 인용 (인라인):
"아버지의 말씀은 진리니이다"(요 17:17)라고 예수님은 말씀하셨습니다.

블록 인용:
> "하나님이 자기 형상대로 사람을 창조하시되..."
> — 창세기 1:27

여러 구절:
마태복음 24장은 마지막 때의 미혹을 경고합니다.
골로새서 3:16은 말씀으로 서로를 가르칠 것을 권면합니다.


인용 형식 (English)
─────────────────────────────────────────────────────────────

Short (inline):
Jesus said, "Sanctify them in the truth; your word is truth" (John 17:17).

Block quote:
> "So God created man in his own image..."
> — Genesis 1:27 (ESV)
```

### 7.3 자주 인용되는 구절

| 구절 | 주제 | 사용 맥락 |
|------|------|-----------|
| 창 1:27 | 하나님의 형상 | 정체성, About |
| 신 6:6-7 | 말씀 전수 | 가정, F.A.M.I.L.Y |
| 요 17:17 | 진리 | 철학, 분별 |
| 골 3:16 | 서로 가르침 | 공동체, 하브루타 |
| 히 10:24-25 | 모임 | 공동체, 보호 |
| 요일 4:1 | 영 분별 | ASI 시대, 질문 |

## 8. 타겟별 톤 조절 | Tone by Audience

### 8.1 교회 리더

```
TONE FOR CHURCH LEADERS
─────────────────────────────────────────────────────────────

톤: 전문적 + 공감
존칭: 높임말 유지
용어: 신학/목회 용어 OK
강조: 실용성, 검증된 결과

예시:
"목회 현장에서 가장 어려운 것 중 하나는
 지속 가능한 부흥의 구조를 세우는 것입니다.
 ChurchThrive는 이 과제를 함께 풀어가는 파트너입니다."
```

### 8.2 가정/부모

```
TONE FOR FAMILIES/PARENTS
─────────────────────────────────────────────────────────────

톤: 따뜻함 + 격려
존칭: 친근하되 예의
용어: 쉬운 용어
강조: 공감, 실용적 팁

예시:
"아이들에게 신앙을 전하고 싶은 마음,
 충분히 이해합니다.
 작은 것부터 함께 시작해볼까요?"
```

### 8.3 청년/개인

```
TONE FOR YOUNG ADULTS
─────────────────────────────────────────────────────────────

톤: 영감 + 도전
존칭: 편안한 높임말
용어: 현대적 표현 OK
강조: 비전, 자기 발견

예시:
"AI가 모든 답을 알려주는 시대,
 당신은 어떤 질문을 가지고 있나요?
 그 질문이 당신을 만들어갑니다."
```

### 8.4 소그룹 리더

```
TONE FOR SMALL GROUP LEADERS
─────────────────────────────────────────────────────────────

톤: 지지 + 실용
존칭: 친근한 높임말
용어: 교회 용어 + 쉬운 설명
강조: 도구, 단계별 가이드

예시:
"매주 구역예배를 준비하는 것,
 쉽지 않으시죠?
 여기 바로 쓸 수 있는 가이드가 있습니다."
```

## 9. 품질 체크리스트 | Quality Checklist

### 9.1 작성 전 확인

```
□ 타겟 오디언스는 누구인가?
□ 이 콘텐츠의 목적은 무엇인가?
□ 어떤 톤이 적절한가?
□ 핵심 메시지는 무엇인가?
```

### 9.2 작성 후 확인

```
□ 브랜드 보이스와 일치하는가?
□ 톤이 상황에 적절한가?
□ 문장이 명확하고 간결한가?
□ 성경 인용이 정확한가?
□ 피해야 할 표현은 없는가?
□ CTA가 명확한가?
□ 희망적인 메시지를 전달하는가?
□ 오타와 문법 오류는 없는가?
```

### 9.3 최종 확인

```
□ 소리 내어 읽었을 때 자연스러운가?
□ 타겟 독자가 이해할 수 있는가?
□ Philosophy.md의 정신과 일치하는가?
□ 행동을 유도하는가?
```
