## 1. CTA 전략 개요 | CTA Strategy Overview

### 1.1 CTA 목표

```
┌─────────────────────────────────────────────────────────────┐
│                      CTA GOALS                               │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  1. 뉴스레터 구독 (Email Collection)                        │
│     → 1차 전환, 지속적 관계 구축                            │
│                                                              │
│  2. 리소스 다운로드 (Lead Magnet)                           │
│     → 가치 제공 + 이메일 수집                               │
│                                                              │
│  3. 상담/문의 (Sales Inquiry)                               │
│     → ChurchThrive 등 유료 서비스 연결                      │
│                                                              │
│  4. 프로젝트 참여 (Product Usage)                           │
│     → COMPASS 진단, MATCH 시작 등                           │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 CTA 퍼널

```
CTA FUNNEL
─────────────────────────────────────────────────────────────

Awareness (인지)
    │
    ▼
┌─────────────────────────────────────┐
│  "더 알아보기" / "철학 읽기"        │
│  → About, Philosophy 페이지로       │
└─────────────────────────────────────┘
    │
    ▼
Interest (흥미)
    │
    ▼
┌─────────────────────────────────────┐
│  "프로젝트 둘러보기"                │
│  → Projects Hub로                   │
└─────────────────────────────────────┘
    │
    ▼
Consideration (고려)
    │
    ▼
┌─────────────────────────────────────┐
│  "무료 가이드 받기" / "뉴스레터"    │
│  → 이메일 수집                      │
└─────────────────────────────────────┘
    │
    ▼
Decision (결정)
    │
    ▼
┌─────────────────────────────────────┐
│  "무료 상담 신청" / "진단 시작"     │
│  → 직접 참여/전환                   │
└─────────────────────────────────────┘
```

## 2. CTA 유형 | CTA Types

### 2.1 Primary CTAs (주요 CTA)

| CTA 텍스트 | 용도 | 위치 | 목표 |
|------------|------|------|------|
| 프로젝트 둘러보기 | 탐색 유도 | Landing Hero | Hub 방문 |
| 지금 시작하기 | 참여 유도 | 프로젝트 페이지 | 직접 참여 |
| 무료 상담 신청 | 리드 수집 | ChurchThrive | 상담 전환 |
| 무료 진단 시작 | 제품 사용 | COMPASS | 진단 시작 |
| 가정예배 시작하기 | 제품 사용 | MATCH | 사용 시작 |

### 2.2 Secondary CTAs (보조 CTA)

| CTA 텍스트 | 용도 | 위치 |
|------------|------|------|
| 철학 알아보기 | 깊이 탐색 | Landing |
| 더 알아보기 | 상세 정보 | 카드, 섹션 |
| 자세히 보기 | 상세 페이지 | 프로젝트 카드 |
| 사례 보기 | 신뢰 구축 | 프로젝트 페이지 |

### 2.3 Tertiary CTAs (추가 CTA)

| CTA 텍스트 | 용도 | 위치 |
|------------|------|------|
| 뉴스레터 구독 | 리드 수집 | Footer, 전용 섹션 |
| 무료 가이드 받기 | 리드 마그넷 | Resources |
| 문의하기 | 연락 | 전체 페이지 |

## 4. CTA 디자인 가이드 | CTA Design Guide

### 4.1 버튼 스타일

```
CTA BUTTON STYLES
─────────────────────────────────────────────────────────────

Primary Button:
┌─────────────────────────────────────┐
│         프로젝트 둘러보기            │
└─────────────────────────────────────┘
• Background: Deep Navy (#1A2B4A)
• Text: White
• Padding: 16px 32px
• Border-radius: 8px
• Font-weight: 600
• Hover: 10% lighter

Secondary Button:
┌─────────────────────────────────────┐
│         철학 알아보기               │
└─────────────────────────────────────┘
• Background: Transparent
• Border: 2px solid Deep Navy
• Text: Deep Navy
• Hover: Filled background

Text Link CTA:
  자세히 보기 →
• Text: Deep Navy
• Underline on hover
• Arrow icon suffix
```

### 4.2 버튼 사이즈

| 사이즈 | Padding | Font | 사용 |
|--------|---------|------|------|
| Large | 18px 40px | 18px | Hero, 주요 CTA |
| Medium | 14px 28px | 16px | 일반 CTA |
| Small | 10px 20px | 14px | 카드 내, 인라인 |

### 4.3 CTA 배치 규칙

```
CTA PLACEMENT RULES
─────────────────────────────────────────────────────────────

1. Above the Fold
   Hero CTA는 반드시 스크롤 없이 보여야 함

2. Visual Hierarchy
   Primary > Secondary > Tertiary
   크기, 색상, 위치로 구분

3. Proximity
   관련 콘텐츠 근처에 배치
   (예: 가격 정보 옆에 구매 CTA)

4. Repetition
   긴 페이지는 중간에 CTA 반복
   단, 과하지 않게 (2-3회)

5. Contrast
   배경과 대비되는 색상 사용
   화이트 스페이스 확보
```

## 5. CTA 카피라이팅 | CTA Copywriting

### 5.1 작성 원칙

```
CTA COPY PRINCIPLES
─────────────────────────────────────────────────────────────

1. 동사로 시작
   ✓ "시작하기", "받기", "발견하기"
   ✗ "시작", "다운로드", "발견"

2. 혜택 암시
   ✓ "무료 가이드 받기" (무료 + 가이드)
   ✗ "제출하기" (혜택 없음)

3. 긴급성/희소성 (적절히)
   ✓ "지금 시작하기"
   ✗ "나중에 하기"

4. 구체적
   ✓ "15분 진단 시작하기"
   ✗ "시작하기"

5. 간결함
   ✓ 2-4 단어
   ✗ 긴 문장
```

### 5.2 CTA 예시 라이브러리

```
EXPLORATION CTAs:
• 더 알아보기
• 프로젝트 둘러보기
• 철학 읽기
• 자세히 보기 →

LEAD MAGNET CTAs:
• 무료 가이드 받기
• 시작 가이드 다운로드
• 무료 자료 받기
• PDF 받기

NEWSLETTER CTAs:
• 구독하기
• 뉴스레터 받기
• 소식 받아보기
• 함께하기

ACTION CTAs:
• 지금 시작하기
• 무료로 시작하기
• 진단 시작하기
• 가정예배 시작하기

CONTACT CTAs:
• 무료 상담 신청
• 문의하기
• 상담 받기
• 연락하기
```

## 6. 타겟별 CTA 전략 | CTA by Target Audience

### 6.1 교회 리더

```
TARGET: 교회 리더
─────────────────────────────────────────────────────────────

Pain Point: "교회 부흥이 막막하다"

Primary CTA: "무료 상담 신청"
• 부담 없는 첫 단계 제시
• "무료" 키워드로 진입 장벽 낮춤

Secondary CTA: "ChurchThrive 소개서 받기"
• 정보 먼저 수집
• 의사결정 지원

Messaging:
"교회의 상황을 듣고 함께 방향을 찾아드립니다"
```

### 6.2 가정 부모

```
TARGET: 가정 부모
─────────────────────────────────────────────────────────────

Pain Point: "자녀 신앙교육 방법을 모르겠다"

Primary CTA: "무료 가이드 받기"
• 즉시 사용 가능한 가치 제공
• 낮은 진입 장벽

Secondary CTA: "가정예배 시작하기"
• 구체적 행동 제안

Messaging:
"오늘부터 시작할 수 있는 가정예배 가이드"
```

### 6.3 청년/개인

```
TARGET: 청년/개인
─────────────────────────────────────────────────────────────

Pain Point: "내 은사가 뭔지 모르겠다"

Primary CTA: "무료 진단 시작하기"
• 즉각적 행동
• 호기심 자극

Secondary CTA: "15분 테스트 해보기"
• 시간 구체화로 부담 감소

Messaging:
"15분 진단으로 하나님이 주신 선물을 발견하세요"
```

## 7. A/B 테스트 계획 | A/B Testing Plan

### 7.1 테스트 항목

```
A/B TEST ITEMS
─────────────────────────────────────────────────────────────

1. CTA 텍스트
   A: "지금 시작하기"
   B: "무료로 시작하기"

2. 버튼 색상
   A: Deep Navy
   B: Living Green

3. 버튼 위치
   A: 중앙 정렬
   B: 좌측 정렬

4. CTA 개수
   A: 1개 (Primary만)
   B: 2개 (Primary + Secondary)

5. 긴급성 메시지
   A: "지금 시작하기"
   B: "오늘 시작하기"
```

### 7.2 테스트 우선순위

| 순위 | 테스트 | 예상 영향 | 난이도 |
|------|--------|-----------|--------|
| 1 | CTA 텍스트 | 높음 | 낮음 |
| 2 | 버튼 색상 | 중간 | 낮음 |
| 3 | CTA 위치 | 중간 | 중간 |
| 4 | 페이지 레이아웃 | 높음 | 높음 |

## 8. 측정 및 최적화 | Measurement

### 8.1 CTA KPIs

| 지표 | 정의 | 목표 |
|------|------|------|
| 클릭률 (CTR) | 클릭 / 노출 | 5%+ |
| 전환율 | 전환 / 클릭 | 10%+ |
| 이탈 지점 | 어디서 이탈하는가 | 최소화 |
| 시간 대비 전환 | 체류 시간 vs 전환 | 상관관계 분석 |

## 9. CTA 체크리스트 | CTA Checklist

### 페이지 게시 전 확인

```
□ Primary CTA가 Above the Fold에 있는가?
□ CTA 텍스트가 동사로 시작하는가?
□ 혜택이 명확하게 암시되는가?
□ 버튼이 시각적으로 눈에 띄는가?
□ 모바일에서도 탭하기 쉬운 크기인가?
□ 클릭 추적이 설정되어 있는가?
□ 랜딩/목적지 페이지가 준비되어 있는가?
```
