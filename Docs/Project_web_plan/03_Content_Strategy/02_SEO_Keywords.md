## 4. On-Page SEO 가이드 | On-Page SEO Guide

### 4.1 메타 태그 템플릿

```html
<!-- 기본 구조 -->
<title>[Primary Keyword] - [Secondary Keyword] | Havruta Project</title>
<meta name="description" content="[Value Proposition + Primary Keyword + CTA] (150-160자)">

<!-- 예시: Landing Page -->
<title>Havruta Project - ASI 시대의 교회 부흥</title>
<meta name="description" content="말씀과 공동체로 더 깊은 사람을 세우는 프로젝트 생태계. 교회 부흥, 가정예배, 은사 발견을 위한 솔루션을 만나보세요.">

<!-- 예시: ChurchThrive -->
<title>ChurchThrive - 교회 부흥 프로그램 | 15일 부트캠프</title>
<meta name="description" content="15일 부트캠프로 교회의 지속 가능한 부흥 체질을 만듭니다. 무료 상담 신청하고 교회 진단 받아보세요.">
```

### 4.2 Heading 구조

```
H1: 페이지당 1개, 주요 키워드 포함
    예: "교회 부흥의 구조를 세웁니다 - ChurchThrive"

H2: 섹션 제목, 보조 키워드 포함
    예: "15일 부트캠프로 시작하는 부흥의 루프"

H3: 하위 섹션, 관련 키워드
    예: "Week 1: 교회 현황 진단"

H4-H6: 세부 항목
```

### 4.3 콘텐츠 최적화

```
CONTENT OPTIMIZATION CHECKLIST
─────────────────────────────────────────────────────────────

□ 첫 100단어 내 주요 키워드 포함
□ 자연스러운 키워드 밀도 (1-2%)
□ 관련 키워드 및 LSI 키워드 사용
□ 내부 링크 2-5개 포함
□ 외부 링크 (권위 있는 출처) 1-2개
□ 이미지 alt 태그에 키워드 포함
□ URL에 키워드 포함 (영문)
□ 최소 300단어 이상 (상세 페이지 800+)
```

### 4.4 URL 구조

```
GOOD URL EXAMPLES
─────────────────────────────────────────────────────────────

✓ /projects/churchthrive
✓ /projects/compass
✓ /resources/family-worship-guide
✓ /blog/church-revival-in-ai-era

BAD URL EXAMPLES
─────────────────────────────────────────────────────────────

✗ /projects/123
✗ /page?id=churchthrive
✗ /자료/가정예배-가이드 (한글 URL 지양)
```
