## 1. 프로젝트 개요 | Project Overview

### 1.1 COMPASS란?

```
┌─────────────────────────────────────────────────────────────┐
│                     CP21L_COMPASS                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  정의:                                                       │
│  성품(Character)과 은사(Gift) 기반 사역 매칭 시스템         │
│                                                              │
│  핵심 기능:                                                  │
│  • 성품 진단 (Character Assessment)                         │
│  • 은사 발견 (Spiritual Gift Discovery)                     │
│  • 사역 매칭 (Ministry Matching)                            │
│  • 성장 로드맵 (Growth Roadmap)                             │
│                                                              │
│  타겟:                                                       │
│  • 사역 탐색 중인 성도                                      │
│  • 은사를 발견하고 싶은 청년                                │
│  • 인재 관리가 필요한 교회                                  │
│                                                              │
│  핵심 가치:                                                  │
│  "기능이 아니라 성품, 경쟁이 아니라 부르심"                 │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 Tagline & Messages

| 요소 | 한국어 | English |
|------|--------|---------|
| Tagline | 당신의 은사와 사역을 찾아드립니다 | Discover Your Gifts, Find Your Ministry |
| Value Prop | 성품과 은사를 진단하고 맞는 사역을 매칭합니다 | Assess your character and gifts, match with the right ministry |
| CTA | 나의 은사 발견하기 | Discover My Gifts |

## 6. 기능적 고려사항 | Functional Considerations

### 6.1 진단 시스템 연동

```
- 진단 시작 CTA 클릭 → 진단 페이지/앱으로 이동
- 결과 저장 → 사용자 계정 연동 (옵션)
- 교회 관리자 대시보드 → 별도 관리 시스템
```

### 6.2 향후 확장

```
- API 연동 (교회 관리 시스템)
- 모바일 앱 연동
- PDF 리포트 생성
```
