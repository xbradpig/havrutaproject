## 4. 다운로드 플로우 | Download Flow

### 4.1 게이트 여부

```
DOWNLOAD GATING STRATEGY
─────────────────────────────────────────────────────────────

Ungated (이메일 불필요):
• 인포그래픽
• 체크리스트 (1페이지)
• 샘플 템플릿

Soft-gated (이메일 선택):
• 짧은 가이드 (5페이지 이하)
• 아티클 PDF

Gated (이메일 필수):
• 종합 가이드 (10페이지 이상)
• 워크북
• 코스 자료
```

## 5. 개발 명세 | Development Spec

### 5.1 데이터 구조

```typescript
interface Resource {
  id: string;
  type: 'download' | 'article' | 'guide';
  title: string;
  description: string;
  thumbnail?: string;
  category: string[];
  tags: string[];
  downloadUrl?: string;
  articleUrl?: string;
  pageCount?: number;
  readTime?: string;
  gated: 'none' | 'soft' | 'hard';
  featured: boolean;
  createdAt: Date;
  updatedAt: Date;
}
```

## 7. SEO 전략 | SEO Strategy

### 7.1 메타데이터

```html
<title>자료실 - Havruta Project | 무료 가이드 & 자료</title>
<meta name="description" content="가정예배 가이드, 소그룹 자료, 리더십 훈련 자료 등 Havruta Project의 모든 무료 자료를 다운로드하세요.">
```

### 7.2 개별 자료 SEO

- 각 자료는 고유 URL 가능 (`/resources/family-worship-guide`)
