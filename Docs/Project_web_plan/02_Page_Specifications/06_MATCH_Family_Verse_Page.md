## 1. 프로젝트 개요 | Project Overview

### 1.1 MATCH_Family_Verse란?

```
┌─────────────────────────────────────────────────────────────┐
│                   MATCH_Family_Verse                         │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  정의:                                                       │
│  가정과 소그룹을 위한 하브루타 기반 성경공부 플랫폼         │
│                                                              │
│  핵심 원리:                                                  │
│  F.A.M.I.L.Y = Faith · Ask · Memorize · Inquire · Learn · Yield │
│                                                              │
│  핵심 기능:                                                  │
│  • 말씀 암송 시스템                                         │
│  • 매칭(동반자) 시스템                                      │
│  • 질문 & 나눔 가이드                                       │
│  • 순종 결단 기록                                           │
│                                                              │
│  타겟:                                                       │
│  • 기독교 가정 (부모)                                       │
│  • 소그룹/구역 리더                                         │
│  • 청소년부/주일학교 교사                                   │
│                                                              │
│  핵심 가치:                                                  │
│  "가정이 작은 교회가 됩니다"                                │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 Tagline & Messages

| 요소 | 한국어 | English |
|------|--------|---------|
