## 1. 프로젝트 개요 | Project Overview

### 1.1 BlueHill21Leader란?

```
┌─────────────────────────────────────────────────────────────┐
│                    BlueHill21Leader                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  정의:                                                       │
│  리더십 훈련과 체계적 지식관리(CMDS)를 통합한 프로젝트       │
│                                                              │
│  핵심 구성:                                                  │
│  1. 리더십 훈련 프로그램 (21가지 성품 기반)                 │
│  2. CMDS 3-레이어 문서 시스템                               │
│  3. Obsidian 기반 지식 관리                                 │
│                                                              │
│  타겟:                                                       │
│  • 교회 리더 (장로, 권사, 집사)                             │
│  • 리더 후보생                                               │
│  • 목회자/사역자                                             │
│  • 신학생                                                    │
│                                                              │
│  핵심 가치:                                                  │
│  "기능이 아니라 성품으로 리더를 세운다"                     │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```
